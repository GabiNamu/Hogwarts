## Run locally:
- First clone the repository into your machine;

```
git clone git@github.com:GabiNamu/Hogwarts.git
```

- Go to the project directory:
```
cd Hogwarts
```
 
- Install dependencies;

```
npm install
```
- Start the server;
```
npm start
```
